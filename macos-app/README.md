# 薯片手 ChipHand

一边吃薯片，一边畅快浏览；不用担心弄脏键盘和触控板。

薯片手是原生 macOS 视觉手势控制 App。MediaPipe Gesture Recognizer、WASM、模型和说明文档全部内置，摄像头画面不保存、不上传；Apple Vision 只作为 MediaPipe 启动失败时的备用引擎。

完整图文说明见 [`docs/user-guide/index.html`](../docs/user-guide/index.html)。

## 普通用户安装

1. 打开 `ChipHand-macOS-1.0.1-universal.dmg`。
2. 把“薯片手.app”拖到“Applications / 应用程序”。
3. 首次若被 macOS 拦截，右键 App 选择“打开”；或到“系统设置 → 隐私与安全性”点“仍要打开”。
4. 允许“摄像头”和“辅助功能”，回到 App 点“开启手势控制”。

不需要安装 Xcode、Node.js、Python 或其他运行环境。免费开源版没有 Developer ID 公证，因此其他用户首次下载后需要做一次安全放行。

## 当前动作

- 严格 OK 手势捏住后上下移动：连续滚动，松开或丢手立即停止。
- 手势呈 OK 状，食指和拇指之间出现黄色点；从屏幕左侧向右越过中线：在 Chrome、Safari、Edge 和夸克返回上一页。
- 从屏幕右侧越过中线：在上述浏览器前进下一页。
- 严格 OK 水平锁定后首次到达中线时：出现 `352pt` 白色线条提醒，表示已可松手。
- 张开手掌向右挥动：向下翻约 75% 屏幕；向左挥动：向上翻约 75% 屏幕。达到水平挥动距离即可，不要求跨过屏幕中线。
- 食指控制点永久开启：在 Chrome、Safari、Edge、夸克中竖起食指移动黄色点；停在目标上后，用拇指与中指捏合直到出现第 2 个黄色点，完成普通左键点击，也可用于播放与暂停控制。
- V 手势：只显示识别状态，暂不执行操作。
- 竖起拇指：显示点赞识别反馈，不执行真实点赞。
- 黄色食指控制点和指针操作永久开启，不提供关闭选项；指针仅支持 MacBook 主屏幕和四个浏览器。
- 开启后默认在整个屏幕显示点击穿透的手部骨架和控制点；关闭骨架后黄色操作点与跨中线闪光仍保留。底层仍使用完整 21 点，显示层每根手指简化为 3 段。
- 实时状态条固定在 Dock 上方默认位置；单击右侧三道杠收到屏幕右边缘，迷你条可上下移动，双击恢复。
- 双击展开状态条可暂停/恢复识别；暂停时圆点变红并显示“已暂停手势控制”。
- 摄像头校准窗口默认关闭，需要检查画面与骨架对齐时可从菜单打开。
- 主控制窗口可选择“右手”或“左手”，默认右手；只有所选手能显示骨架并执行动作。
- 菜单中的“测试系统下翻”可绕过手势识别，直接验证辅助功能和系统滚动是否正常。
- App 同时出现在 Dock/程序坞和菜单栏；关闭控制窗口不会停止后台手势识别。

## 开发与打包

建议安装完整 Xcode，然后执行：

```bash
cd macos-app
swift test
swift run GestureControlCoreChecks
./scripts/build-app.sh
open build/ChipHand.app
```

稳定安装到“应用程序”：

```bash
./scripts/install-app.sh
```

生成可分发的 Universal DMG 与 ZIP：

```bash
./scripts/package-release.sh
```

产物位于 `macos-app/releases/`。固定 bundle ID 为 `com.huyee.chiphand`；品牌切换后的第一次安装需要重新允许两项权限，后续同一身份的小版本更新不应反复授权。
