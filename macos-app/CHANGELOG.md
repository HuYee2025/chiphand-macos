# Changelog

## 0.1.1 - 2026-07-15

- 修复辅助功能权限已经打开但 App 仍反复跳回系统设置的循环：启动按钮不再自动打开设置，App 每秒刷新权限，待授权生效后自动继续启动。
- 新增默认开启的悬浮识别测试窗，实时显示镜像摄像头、手部骨架、捏合连线和识别状态。
- 新增稳定安装脚本，将原型安装到 `/Applications/GestureControl.app`，避免退出后找不到入口。
- 新增标准 `XCTest` Target，覆盖捏合稳定、停止、翻页方向和取消状态。
- 更新本地构建号，安装新二进制后重新建立一次辅助功能授权，后续不重建 App 时不再反复失效。

## 0.1.0 - 2026-07-15

- 新增原生 macOS 菜单栏手势控制原型。
- 使用 AVFoundation 获取摄像头画面，使用 Apple Vision 检测单手关键点。
- 左挥上翻、右挥下翻约 75% 屏幕；捏合稳定 80ms 后上下拖动连续滚动。
- 通过 Core Graphics 将滚动事件发送给前台 App，不移动鼠标、不发送点击或键盘事件。
- 新增摄像头与辅助功能权限引导、两项灵敏度设置和可选调试预览。
- 新增纯 Swift 核心检查程序、本地 `.app` 打包与 ad-hoc 签名脚本。
