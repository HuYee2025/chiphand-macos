# 生产蓝图

## 项目一句话

使用一只手实时控制《火星先遣队》无限循环黑洞的独立 Web 互动 Demo。

## 目标用户/使用场景

- MacBook Air 摄像头实机研究与互动展示。
- 带摄像头的 Windows Chrome/Edge。
- 支持摄像头权限的手机 Safari/Chrome。

## 核心体验

- 手掌位置控制上下左右方向。
- 手掌左右倾斜控制旋转速度：保持倾斜时持续旋转，回正后停在当前角度。
- 手掌在画面中的尺寸用于估算距离：初次出现约 0.6 秒校准为 1×，靠近加速、远离减速，范围限制在 0.45×～2.4×。
- 张手继续、握拳暂停。
- 摄像头小窗实时显示 21 点骨架；左手红色、右手蓝色。

## 功能/内容结构

- 全屏 Three.js 无限循环黑洞。
- 首次摄像头授权引导。
- 摄像头与关键点叠加小窗，可隐藏。
- 鼠标、触控和空格键作为失败降级输入。

## 视觉与风格

- 深色、机械、有机虫洞；沿用原始黑洞原型的象牙色和黑色结构。
- UI 使用克制的实验室/航天研究界面，不遮挡核心画面。

## 技术约束

- 技术栈为 Vite、TypeScript、Three.js、MediaPipe Tasks Vision。
- 摄像头必须在 HTTPS 或 localhost 使用。
- HandLandmarker 自适应 24–30 FPS，GPU 优先、CPU 备用；3D 渲染目标 60 FPS。
- 关键点逐点平滑，短暂遮挡保留最后可信姿态约 220ms。

## 不做什么

- 不回写《火星先遣队》原项目。
- 第一版不增加角色、潜艇、关卡、多人、音效和账号系统。
- 暂不训练自定义手势模型。
